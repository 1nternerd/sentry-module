## Setup
- Add `@nuxtjs/sentry` dependency using yarn or npm to your project
- Add `@nuxtjs/sentry` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    '@nuxtjs/sentry',

    // With options
    ['@nuxtjs/sentry', { /* module options */ }],
 ]
}
```
